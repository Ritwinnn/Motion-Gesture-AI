# Motion Gesture AI

A machine learning-based motion gesture recognition system built using Edge Impulse and sensor data.

This project detects and classifies four hand gestures:

- Up-Down
- Shake
- Idle

## Live Demo

Try the working Motion Gesture AI web application here:

🚀 **Motion Gesture AI Web App:** [Live Demo Link Here]
https://motion-gesture-c2c4e.web.app/

## Project Overview

The system uses accelerometer sensor data to recognize hand movements in real time.  
The trained model can process gesture inputs and classify movements for real-world applications.

## Features

- Real-time gesture recognition
- Live web-based demo
- Edge Impulse model integration
- Sensor-based motion detection
- Lightweight deployment

## Dataset

The dataset consists of:

- **Up** → Upward and Downward hand movement
- **Shake** → Repetitive shaking motion
- **Idle** → No movement / resting state

## Tech Stack

- Python
- Edge Impulse
- Machine Learning
- Arduino / ESP32
- Accelerometer Sensor

## Workflow

1. Collect motion data  
2. Upload data to Edge Impulse  
3. Label gestures  
4. Train the model  
5. Test accuracy  
6. Deploy the model  

## Applications

- Smart home control  
- Gesture-based UI systems  
- Gaming controls  
- Robotics  
- Accessibility tools
